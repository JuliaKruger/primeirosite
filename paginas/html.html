Exemplos de Formatação de Texto

  • texto negrito - <b>texto negrito</b>
  • texto forte - <strong>texto forte</strong>
  • texto enfatizado - <em>texto enfatizado</em>
  • texto italico - <i>texto italico</i>
  • texto grifado - <mark>texto grifado</mark>
  • texto grande - <big>texto grande</big>
  • texto pequeno - <small>texto pequeno</small>
  • texto taxado - <del>texto taxado</del>
  • texto inserido - <ins>texto inserido</ins>
  • texto sublinhado - <u>texto sublinhado</u>
  • esse é umtexto subscrito - <sub>texto subscrito</sub>
  • esse é umtexto sobreecrito - <sup>texto sobreescrito</sup>

Exemplo de Tabelas

Tabela com 2 linhas e 2 colunas:
linha 1, coluna 1 linha 1, coluna 2
linha 2, coluna 1 linha 2, coluna 2
<table border=1 width=100%>
<tr>
<td>linha 1, coluna 1</td>
<td>linha 2, coluna 1</td>
</tr>
<tr>
<td>linha 2, coluna 1</td>
<td>linha 2, coluna 2</td>
</tr>
</table>

Tabela com 2 linhas de dados, 1 linha de cabeçalho e 2 colunas:
cabeçalho 1 cabeçalho 2
linha 1, coluna 1
 linha 1, coluna 2
linha 2, coluna 1 linha 2, coluna 2
<table border=1 width=100%>
<tr>
<th>cabeçalho 1</th>
<th>cabeçalho 2</th>
</tr>
<tr>
<td><center>linha 1, coluna 1</center></td>
<td align="right">linha 1, coluna 2</td>
</tr>
<tr>
<td>linha 2, coluna 1</td>
<td>linha 2, coluna 2</td>
</tr>
</table>

Tabela com exemplo de colunas mescladas usando colspan:

</table>

Tabela com exemplo de linhas mescladas usando rowspan:
